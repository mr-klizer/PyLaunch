Как установить Python — пошаговая инструкция
1
Выберите версию и нажмите кнопку скачать
Рекомендуем Python 3.11 — самая стабильная версия, с ней работает большинство программ. Нажмите нужную кнопку ниже — откроется страница загрузки на официальном сайте python.org в вашем браузере.
2
На сайте найдите файл для Windows
Прокрутите страницу вниз до раздела «Files». Найдите строку Windows installer (64-bit) и нажмите на неё — начнётся скачивание файла с расширением .exe весом около 25 МБ.
Если у вас 32-битная Windows (редкость) — выбирайте Windows installer (32-bit)
3
Запустите скачанный файл
Откройте скачанный файл (python-3.11.x-amd64.exe). Windows может спросить разрешение — нажмите «Да».
Важно! В первом окне установщика обязательно поставьте галочку «Add Python to PATH» — без неё программы не найдут Python.
4
Нажмите «Install Now»
Установка занимает 1–2 минуты. После появится сообщение «Setup was successful» — Python установлен.
5
Вернитесь в PyLaunch и нажмите «Обновить список»
После установки нажмите кнопку «Обновить список» в правом верхнем углу этой страницы. Новая версия Python появится в списке выше — выберите её кнопкой «Выбрать».
6
Подготовьте программу к запуску
Теперь перейдите в «Каталог программ», найдите нужную программу и нажмите «Подготовить» — PyLaunch автоматически установит все необходимые компоненты.
