# Подробнее о версиях (файлы установки ниже)

## Релиз Pylaunch 1.0.0
Эта самая первая версия нашей программы, опубликованной в общий доступ. На момент публикации все известные нам баги были пофикшены.

### Функционал:
- **Реализовано 3 способа вставки программы** - Вы можете вставить вашу программу, указав папку, архив или одиночный файл. В случае с архивом, программа сама распакует архив и сохранит себе проект в корневую папку (по умолчанию C:\Users\\~\PyLaunch Projects).

- **Реализована возможность резервного копирования** - Если вы добавляете вашу программу из одиночного файла или папки, вам будет предоставлена возможность выбрать (пункт 1 и 2 ниже). При вставки программы из архива, вам не будет предложено никаких вариантов и программа автоматически выполнит пункт 1. 
```
    1. Создать резервную копию (программа дублирует вашу программу себе 
       в корневую папку и будет запускать именно дубликат)
    2. Использовать исходный путь (программа не станет ничего 
       дублировать и лишь запомнит расположение вашей программы для 
       дальнейшего запуска)
```

- **Реализован выбор интерпритатора** - программа содержит в себе подробную инструкцию установки интерпритатора, предлагает самые стабильные версии, способна удалить интерпритатор по одной кнопке, самостоятельно находит интерпритатор на вашем ПК.

Далее мы рассмотрим функционал, созданный для управления программами

- **Реализованы дополнительные параметры по запуску программы** - На карточке программы есть переключатель "запуска программы с терминалом" и "автоматическим закрытием терминала при завершении работы программы". Данные настройки необходимы, когда добавленная вами программа не имеет собственного интерфейса или есть необходимость в использовани консоли.

- **Реализовано автоматическое чтение README, requiments и banner файлов** - Если в программе найдены подобные файлы, программа встроит их в venv и карточку проекта. К вашему сведению, banner - это просто фотография обложки программы.

- **Реализован просмотр README, редактирование баннера программы, изменение файла запуска, открытие рабочей папки, удаление программы из списка** - Чтобы это сделать, нажмите на кнопку "..." в углу карточки одной из программ. При удалении зарезервированной программы, вам предложат удалить только из списка (резервная копия остается на ПК в корневой папке), либо полностью (резервная копия удаляется, но приложение никак не затронет вашу исходную программу). Также приложение поддерживает формат файлов типа .md

- **Автоматическое чтение ошибок** - Приложение уже умеет понимать ошибку нехватки конкретной библиотеки и через свой интерфейс предложит установить ее. В дальнешем планируется добавить больше типов ошибок, понимаемых приложением.
